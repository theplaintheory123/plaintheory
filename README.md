# Plantheory



A modern, visually stunning waitlist landing page for PlainTheory - a platform for career guidance, startup support, co-founder matching, and idea development. Built with Next.js, TypeScript, Tailwind CSS, and Supabase.


## ✨ Features

- **🎨 Modern UI/UX** — Glassmorphism effects, smooth animations, and gradient designs
- **📱 Fully Responsive** — Optimized for desktop, tablet, and mobile devices
- **🔐 Supabase Integration** — Store waitlist entries (name, email, description) securely
- **⚡ Real-time Feedback** — Loading states, success messages, and error handling
- **🎭 Smooth Animations** — Framer Motion powered scroll-triggered animations
- **📊 Live Activity Feed** — Dynamic activity indicators in hero section
- **🎯 SEO Optimized** — Semantic HTML and meta tags
- **🚀 Performance Optimized** — Lazy loading and optimized assets

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **Next.js 14** | React framework with App Router |
| **TypeScript** | Type safety and better developer experience |
| **Tailwind CSS** | Utility-first styling |
| **Framer Motion** | Advanced animations and scroll effects |
| **Supabase** | Backend database and API |
| **React Hooks** | State management and effects |
